Restaurant Web-App usingAngular 6.0

Angular 6.0, Typescript, Node.Js, Reactive Programming

This was my first angular project guided by prof. Jogesh from Hong Kong University.I completed this project through Angular Front-end Framework course on Coursera.

This Project contains basically all the basics and intermediate knowledge about angular. From Angular Material and Angular Flex-Layout for responsive UI design to omponents, directives and services. Also various use of data binding, Angular router and its use for developing single-page applications. Also I have used both template-driven forms and reactive forms.

While for the server side you will see the use of node.js for client-server communication through the HTTP client and the use of REST API on the server side.And finally a little use of angular animations and angular testing rounds.
