Real Time Chat App WhatsApp-web(Clone)

React js, Firebase, Context Api, Redux

This is as Title suggested a Whats-app Web application clone project.Most brilliant feature about this app is you can real time chat from this and that too with google authentication.

This app uses React js as frontend framework while for backend and deployment and for real time database it used Firebase. Firebase is very powerfull tool as it uses NOSQL database and authentication with google is also very easy through firebase.I used React Context Api(Hooks) and as well as a little bit of redux js in this app.
